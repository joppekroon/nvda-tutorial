Focus Mode

We have already seen that NVDA uses some single keys for easy navigation. However, when you want to fill in a form or arrow right to open a submenu you need those keys to do something else.

Fortunately, when you TAB to a form field or properly coded custom control, NVDA will automatically switch to 'Focus mode'. In this mode NVDA will allow the browser to handle all key strokes, except for NVDA + SPACE which will re-activate the original 'Browse mode'.

When 'Focus mode' activates you'll hear a 'chunk' sound, and a 'boop' sound is played when 'Browse mode' activates.

Playground

Tab to the test field, and use NVDA + SPACE to force NVDA back to 'Browse mode', so you can use SHIFT + H to go back to the 'Playground heading'.
