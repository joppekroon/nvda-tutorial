Table Navigation Tip

You can navigate between tables with T, then use ↓ to reach the table proper.

When in the table use CTRL + ALT + ↓, ↑, →, and ← to navigate the cells.

Browser Keyboard Shortcuts
Command Action
TAB Go to next interactive element
SHIFT + TAB Go to previous interactive element
ENTER Activate interactive element

NVDA Keyboard Shortcuts
Command Action
CTRL Stop

SHIFT + H Go to previous heading

SHIFT + 1 - 6 Go to previous heading of level
↓ Reads the next line

→ Reads the next character
← Reads the previous character
CTRL + ↓ Reads the next word
CTRL + ↑ Reads the previous word
NVDA + ↓ Start reading from current line
NVDA + ↑ Reads the current line

NVDA + S Toggle speech between 'on', 'off', and 'beeps'
T Go to next table
SHIFT + T Go to previous table
CTRL + ALT + ↓ Cell below (when in a table)
CTRL + ALT + ↑ Cell above (when in a table)
CTRL + ALT + → Cell to the right (when in a table)
CTRL + ALT + ← Cell to the left (when in a table)

NVDA key

The NVDA key can be set to be Insert and/or Caps Lock in the NVDA settings.
