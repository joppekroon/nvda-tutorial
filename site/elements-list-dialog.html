Elements List Dialog

If you press NVDA + F7, NVDA will show a popup with a list of all the links, headings, form fields, buttons, and landmarks of the page.

Caret browsing

Sometimes, the browser will ask to enable Caret browsing, just select 'Cancel'.

warning Do not check the box to 'never ask again'.

If the Caret browsing popup keeps appearing, you may need to restart your browser.

If you are unfamiliar with landmarks, these are the larger sections of a page, offering screen reader users a coarser but quicker way of navigating a page than jumping through all the links, headings, etc.

Try it out, you should be able to see a few landmarks on this page.

You can navigate the Elements List dialog using TAB, SHIFT + TAB, the arrow keys ↓ ↑ → ←, and ENTER.
